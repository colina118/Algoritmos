Para poder utilizar el codigo se debe de abrir el archivo Tarea5.pro en QT.

Despues compilar y correr el proyecto.
Va a salir una ventana con un polígono, del lado derecho inferior hay una comboBox donde puedes seleccionar que figura deseas
utilizar y despues de seleccionarla puedes realizar las diferentes transformaciones para cada figura y aunque ya hay un valor
predeterminado en cada casilla puedes modificarlo para alterar las transformaciones.
